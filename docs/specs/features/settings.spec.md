# Feature: Settings Page
**Status:** Approved
**Last updated:** 2025-02-24

## Overview
Users can manage their account settings.

## User Stories
- [ ] As a user, I can change my email
- [ ] As a user, I can change my password
- [ ] As a user, I can update my profile info (name, bio, website, avatar)
- [ ] As a user, I can delete my account

## Acceptance Criteria
- Password change requires current password confirmation
- Account deletion is permanent, requires password confirmation
- Profile changes take effect immediately

## Out of Scope (v1)
- Notification preferences
- Privacy settings
- Linked accounts
